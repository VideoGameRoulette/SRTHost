About
SRTHost is a .Net Core application for helping facilitate memory scanning and output for universal speedrunning tools.
Developed By: Squirrelies
Contributors: VideoGameRoulette
Current Version: 2.0.5.0
How To Install SRTHost
Install .Net Core Runtime 3.1.X
  • go to the download link provided above for ".Net Core Runtime" and install it to computer it is required to run this application.
    • Install SRTHost
  • go to the download link provided above for "SRTHost" and extract it to any folder example lets make a folder called SRTHost on the desktop and drag the all items from the zip file into that folder.
    • Install Plugins
  • go to the download link provided above for "Plugins" and take the plugin folder and drag it into the new SRTHost folder we created on the desktop.
    • How To Install VGR's WinForms Plugin
    Install VGR's Plugin
  • go to the download link provided above for "VGR Plugin" and take the SRTPluginVGRUIRE3WinForms folder and drag it into the new plugin folder inside the SRTHost folder we created on the desktop and you can delete the SRTPluginUIRE3WinForms folder since you'll be using the new plugin.
  • take note this plugin only shows Nemesis' HP on the application.
    • How To Install VGR's OBS Widgets
    Install Widgets
  • go to the download link provided above for "OBS Widgets" and take the widgets folder and drag it into the SRTHost folder we created on the desktop.
